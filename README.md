# HR Analytics Dashboard

## Project Overview
This project analyzes employee workforce data using Microsoft Excel. The dashboard helps track employee demographics, attrition trends, and HR performance metrics.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Conditional Formatting

## Key KPIs
- Total Employees
- Attrition Rate
- Average Age
- Department-wise Analysis

## Key Insights
- Identified attrition trends.
- Analyzed employee distribution across departments.
- Tracked workforce demographics.
- Generated HR insights for decision-making.
